# Efetuar ping de um dispositivo

A ferramenta Ping no Portal do cliente do IBM Cloud solicita um eco do servidor selecionado. O ping é usado para verificar alguma
latência em links de comunicação e assegurar que o servidor especificado esteja ativo. Solicitações de ping podem ser enviadas para o
IBM Cloud e dispositivos externos. Siga estas etapas para efetuar ping de um dispositivo:

1. Navegue para a interface **Ferramentas de rede** no
[Portal do cliente ![Ícone de link externo](../../icons/launch-glyph.svg "Ícone de linkexterno")](https://control.softlayer.com/).

* Insira o nome do servidor no campo **Ping**.
* Clique no botão **Ping**. Seus resultados aparecerão no lado direito da tela.
