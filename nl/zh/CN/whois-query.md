# 执行 Whois 查询

通过 IBM Cloud 客户门户网站上的 Whois 查询工具，您可以找到域注册信息，包括联系人信息、名称服务器和域状态。执行以下步骤来执行 Whois 查询：

1. 浏览至[客户门户网站 ![外部链接图标](../../icons/launch-glyph.svg "外部链接图标")](https://control.softlayer.com/) 上的**网络工具**界面。
* 在 Whois 字段中输入要查询的域名。
* 选择 Whois 按钮。相应的结果会显示在屏幕右侧。
