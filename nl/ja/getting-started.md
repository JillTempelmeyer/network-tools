# Network Tools の開始

Network Tools を使用すると、顧客はグローバル登録コミュニティー内の顧客独自のサブネットと IP を管理でき、データ・センター間の速度と待ち時間を決定できます。
 

以下のツールが含まれます。
 * RIR の登録および更新のツール
 * RWHOIS の登録および更新のツール
 * 速度テスト・ツール
 * Looking Glass ツール
